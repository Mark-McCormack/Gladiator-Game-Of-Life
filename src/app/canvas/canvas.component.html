Gladiator Game of Life

Welcome to the Gladiator Game of Life, an interactive simulation inspired by Conway’s Game of Life. This project explores how two competing groups of cells evolve and battle for survival under predefined rules. You can control the board manually or let it run automatically using the AI-driven simulation.

How Does It Work?

The game is based on a grid where each cell can be either alive or dead. The cells follow simple survival rules:

  • Cells with too few neighbors die from isolation.
  • Cells with too many neighbors die from overpopulation.
  • Dead cells can become alive if surrounded by the right number of living neighbors.

In this version, the board is split into two competing teams: Red and Blue. Each team fights for dominance based on the rules of cellular automata.

Game Controls

Use the following buttons to interact with the game:

  • Set the Board! – Generates a new randomized board with Red and Blue cells.
  • Move the Board! – Advances the game by one step.
  • Loop the Board! – Starts or stops an automatic loop that continuously updates the board.
  • Wipe the Board! – Clears the board, resetting it for a new simulation.

AI-Powered Features

In addition to manual control, the project includes an AI model that can analyze board states and predict potential outcomes.

  • AI Search Depth – Adjusts how many turns ahead the AI should analyze.
  • Cell Spawn Rate – Controls how densely populated the board is at the start.
  • Train AI – Runs a machine learning model to improve predictions.
  • Predict Outcome – Uses AI to estimate which team is likely to win.

AI Training Progress

When the AI is training, a progress bar will appear, showing the status. Training logs are available in the console for debugging.

Enjoy the simulation and may the best team survive!
